## DRM功能


SDK的DRM服务由类DRMServer实现，在使用SDK的DWMediaPlayer播放加密视频时，需启动此服务并配置DWMediaPlayer。具体使用方法如下，

DRMServer drmServer = new DRMServer();
drmServer.start();

同时，需要设置MediaPlayer的DRM服务端口号。

int drmServerPort = drmServer.getPort();
player.setDRMServerPort(drmServerPort);

当播放已下载的加密视频时，需调用setDRMVideoPath()方法设置视频文件路径和Context（上下文信息）。

String path="/20140505.pcm";
player.setDRMVideoPath(path, context);

注意：DWMediaPlayer只识别文件扩展名为“.pcm”的加密视频。如果需要播放已下载的加密视频，请使用“.pcm”作为文件的扩展名。

在调用prepareAsync()前，调用DRMServer的reset()方法重置drmserver的设置。

...
drmServer.reset();
player.prepareAsync();
...

当退出当前页面时，可以调用DRMServer的disconnectCurrentStream()方法停止当前流。

drmServer.disconnectCurrentStream();

当应用退出时，可以调用DRMServer的stop()方法停止服务。

drmServer.stop();


DRM v6配置

该配置只有开通drm v6版本的用户需要进行配置，v4版本的用户可以略过本节内容。

drm_demo需要把MainActivity的initDWStorage()这行代码注释取消;

  1. 拷贝drm_v6_solibs文件夹里的内容到libs目录下（Android Studio用户拷到jniLibs目录下）
  2. 实现DWSdkStorage接口
  3. 调用DWStorageUtil.setDWSdkStorage()方法把第一步实现的接口实例设置进来 代码如下：
DWSdkStorage mDWSdkStorage = new DWSdkStorage() {
  @Override
  public void put(String key, String value) {
    ……
  }

  @Override
  public String get(String key) {
    ……
  }
};

DWStorageUtil.setDWSdkStorage(myDWSdkStorage);

注：请在app入口初始化设置此功能，其中第2、3步在需要进行本地播放的时候必须配置。


如何使用DRM Demo


DRM Demo是使用SDK对接CC视频DRM服务的示例源码，目前支持Android4.0及以上版本。Demo的设计旨在展示SDK DRM功能的使用方法，如果希望应用获得更好的使用体验，可根据需求自行更改。

在编译安装Demo前，请按照以下操作进行配置：

  1. 设置帐户信息，在ConfigUtil类中配置USERID（帐户ID）和API KEY。
public final static String API_KEY = "API KEY";
public final static String USERID = "123456";
  1. 添加播放列表，在PlayFragment类中配置playVideoIds（可播放的视频ID）。
//TODO 待播放视频ID列表，可根据需求自定义
private String[] playVideoIds = new String[] {
    "123456",
    "234567",
    "345678", 
    "111111"};
  1. 添加下载列表，在DownloadFragment类中配置downloadVideoIds（可下载的视频ID）。
//TODO 待下载视频ID，可根据需求自定义
private String[] downloadVideoIds = new String[] {
    "123456",
    "234567",
    "345678", 
    "111111"};
